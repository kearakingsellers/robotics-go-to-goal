# Differential Drive Robot - Go To Goal Simulation

## Overview
This project implements a kinematic simulation of a differential drive robot using proportional control to autonomously navigate to a target location.

## Features
- Differential drive kinematics
- Proportional feedback controller
- Angle normalization
- Trajectory visualization using matplotlib

## Technologies Used
- Python
- NumPy
- Matplotlib

## How to Run
pip install -r requirements.txt
python differential_drive_sim.py
